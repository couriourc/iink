# 基于 Tauri React Pixi 实现的桌面标记软件

## 软件功能
实现桌面标记，以及笔记共享

## 领域划分
- 前端
  - Canvas 用户交互体系
    - 基本的压感和不同种类的笔以及其基本功能
  
  - 交互体系
    - 开发上：快捷键体系，不应该侵入代码基本机构，而是使用注册的方式，比如使用 装饰器
    - 用户方向：合理的快捷键
  
  - 状态机
    - 笔迹的撤销备忘录状态机
      - 最大保存量
      - 最大撤销部属
      - 崩溃后修复
      - 笔迹共享的复原

  - 导出与记录
    - PDF 式导出
    - IMG 式导出


- 后端
  - 全局命令的监听
  - 桌面穿透
